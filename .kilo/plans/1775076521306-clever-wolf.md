```cpp
// Contiguous guard
size_t row_bytes_k = ggml_row_size(K->type, K->ne[0]);
size_t row_bytes_v = ggml_row_size(V->type, V->ne[0]);

GGML_ASSERT(K->nb[1] == row_bytes_k && "non-contiguous K not supported for 2-pass");
```

**Tek launch, tek grid, tek reduction path.** Ayrı kernel YOK, ayrı max YOK, ayrı sum YOK.

```cpp
// Causal mask: base_kv dinamik olmalı (hardcoded YASAK)
```

## 5 Kritik Tuzak (FINAL KONTROL LİSTESİ)
## ══════════════════════════════════════════════════════

### 1. Softmax: Tek Launch Zorunlu 🔴

```
YANLIŞ: ayrı kernel ile m_low, ayrı kernel ile m_high
DOĞRU:  tek kernel, tek grid:
  - threadIdx low ve high buffer'larını sırayla dolaşır
  - tek m = max(max_low, max_high) — aynı warp reduction path
  - tek sum = Σ exp(logits_low - m) + Σ exp(logits_high - m)
  - tek normalize: w_i = exp_i / sum

Sebep: ayrı reduction path → FP drift → ranking değişir → reasoning çöker
```

### 2. Causal Mask: base_kv Dinamik 🔴

```
YANLIŞ: kv_pos = i  (hardcoded 0)
DOĞRU:  kv_pos = base_kv + i  (cache'in gerçek write pozisyonu)
        kv_pos_high = base_kv + n_low + i
        if (kv_pos > query_pos) logits = -INFINITY

base_kv kaynağı: sliding window, kv cache shift, batch decode durumuna göre
hardcode YASAK
Sebep: yanlış offset → model geleceği görür (hallucination) veya geçmişi görmez (context loss)
```

### 3. Stride: Contiguous Guard 🔴

```
YANLIŞ: K_high = K + n_low * nb11  (her zaman)
DOĞRU:  GGML_ASSERT(k->nb[1] == row_size)  — contiguous kontrolü
        K_high = (const char*)K + n_low * row_size

Sebep: turbo block'lar farklı boyut, alignment farklı → byte offset yanlış → silent corruption
```

### 4. V Accumulation: FP32 Forced 🟡

```
YANLIŞ: half acc = 0.0h; acc += weight * V[i];  (FP16)
DOĞRU:  float acc = 0.0f; acc += (float)weight * dequant_to_fp32(V[i]);  (FP32)

Sebep: uzun context'te error birikir → kalite %90'dan %60'a düşer (fark etmezsin)
```

### 5. Scale: Manuel Uygulanmalı 🟡

```
Orijinal fused kernel scale'i içeride yapar.
Sen KQ'ları ayırdığın için scale'i her iki pass'e de MANUEL geç.
Çarpım sonrası, masking ve softmax ÖNCESİ uygulanmalı.
```
